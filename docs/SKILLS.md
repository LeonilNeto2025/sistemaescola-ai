# Skill do Agente

## Skill 01

Diretor

Responsável por:

- Cadastrar Funcionários da Secretaria
- Acessar relatórios gerenciais
- Realizar backup
- Acessar logs de acesso

---

## Skill 02

Funcionario da Secretaria

Responsável por:

- Cadastrar alunos
- Cadastrar professores
- Registrar matrículas
- Cadastrar notas
- Cadastrar disciplinas
- Cadastrar cursos
- Cadastrar turmas
- Gerar histórico escolar dos alunos
- Acessar relatórios
- Alterar informações cadastradas

---

## Skill 03

Professor

Responsável por:

- Cadastrar cursos
- Cadastrar disciplinas
- Cadastrar notas
- Relatórios de turmas
- Alterar informações cadastradas

---

## Skill 04

Aluno

Responsável por:

- Acessar suas notas
- Gerar seu relatório escolar
