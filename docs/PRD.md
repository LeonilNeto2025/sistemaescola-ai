# Product Requirements Document

## Nome

Sistema Escolar

## Problema

Dificuldade do diretor, funcionários da secretaria, professores e alunos de gerenciar rotinas administrativas de uma escola de forma precisa e organizada.

## Público-alvo

- Diretor
- Funcionários da secretaria
- Professores
- Alunos

## Objetivos

- Cadastrar aluno
- Cadastrar funcionário da secretaria
- Cadastrar curso
- Cadastrar disciplinas
- Cadastrar notas
- Cadastrar matrícula
- Cadastrar turma
- Validar CPF dos alunos no cadastro
- Validar endereço dos alunos no cadastro
- Validar idade do aluno no cadastro
- Validar celular com máscara de entrada
- Registrar matrícula dos aluno com o curso
- Permitir consultar as informações cadastradas em relatórios
- Permitir editar as informações cadastradas
- Gerar histórico escolar com base nas informações do aluno,  matrícula, curso, turma, disciplinas, notas, professores e última atualização no formato PDF  
- Armazenar logs de acesso e alterações
- Realizar backup das informações
- Tela de login com os seletores de perfil: Diretor, Funcionário da Secreataria, Professor e Aluno
- Tela inicial diferente para cada perfil
